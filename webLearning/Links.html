To link to other areas/sections on this page that have an ID :
  <a href="#NameOfID">Decription Text eg Go to Email section<a>

eg.

Note...
It MUST be preceeded by a '#' in the href command (above).
The place to move to has to have an 'id="EmailSection"' set

Linking to other Websites

  • Google's Website
  • Yahoo News Page

To link to other Websites use :
  <a href="NameOfWebsite">Decription Text eg Google's Website<a>
eg
  <a href="https://www.google.com/">Google's Website<a>

To open in a NEW TAB add the   target="_blank"  command - eg...
  <a href="https://www.google.com/"
    target="_blank">Google's Website<a>

Note... You need the HTTPS://

Linking to Send an Email

  • Send Blank Email to gazzaspc@gmail.com
  • Send Email and Subject Only
  • Send Email and Subject & Body Only
  • Send Email and Subject/Body/CC & BCC

To send emails use :
  <a href="mailto:gazzaspc@gmail.com">Decription Text eg Send Email to Gazzaspc@gmail.com<a>
eg
  <a href=""mailto:gazzaspc@gmail.com">Send Email to Gazzaspc@gmail.com<a>

If you want to include CC/BCC/Subject Line and Body information you can do this BUT
the info MUST be in a certain format.

The FIRST command after the 'mailto email address' MUST start with ?
Each of the other details MUST start with &
Eg.
<a href="mailto:gazzaspc@gmail.com
?cc=secondemail@example.com, anotheremail@example.com
&bcc=lastemail@example.com
&subject=Mail from our Website
&body=Dear Charlie Edgar %0D%0AWhere are you."

Also Note...
  If more than 1 email address is req'd in CC or BCC they must be comma separated.
  If you include the BODY information then it will NOT include the SIGNATURE*
    unless you add that in the body of your text information.
  If you DO NOT include the BODY information then it WILL include the SIGNATURE*
  *(If you have a SIGNATURE set up in your emails)

  ...and %0D%0A is a CRLF to put 'Where are you.' on a new line.
